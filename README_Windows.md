# 微信视频号下载器 - Windows 版

## 安装步骤

1. 解压本压缩包
2. 右键点击 `install.bat`，选择 **"以管理员身份运行"**
3. 等待安装完成

## 使用方法

1. 双击桌面上的 **"微信视频号下载器"** 快捷方式启动
2. 首次运行会提示安装证书，选择 **"是"**
3. 打开 **微信电脑版**
4. 找到想下载的视频号视频
5. 点击进入视频，视频下方会出现 **下载按钮**
6. 点击下载即可

## 卸载

运行安装目录中的 **"卸载.exe"** 即可卸载

## 常见问题

**Q: 下载按钮不出现？**
A: 确认程序正在运行，刷新视频页面重试

**Q: 提示证书安装失败？**
A: 以管理员身份重新运行程序

**Q: 如何查看程序是否运行？**
A: 观察命令行窗口，显示"代理服务启动成功"即表示正常运行
